# Linguagens Formais e Autômatos

Repositório de estudos da disciplina de Linguagens Formais e Autômatos (LFA), com implementações em Python de autômatos finitos determinísticos (AFDs) e exercícios propostos em aula.

## Tecnologias

- Python 3
- Dataclasses

## Estrutura

- `src/`: códigos dos exercícios
- `tests/`: casos de teste e exemplos de uso

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/EnricoNSilva/Linguagens-Formais-Automatos.git

2. Execute o exercício desejado:
    ```bash
    python AFD.py
